# The Spiral

An interactive 3D data essay experience. Walk along a spiral path and discover personal data essays about taste, culture, and behavior.

Built by [@thedatastudioarchive](https://substack.com/@thedatastudioarchive).

## Stack

- **Next.js 14** (App Router)
- **React Three Fiber** + **Three.js** — 3D spiral environment
- **@react-three/drei** — HTML overlays in 3D space
- **TypeScript**
- **Tailwind CSS v4**

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).


## Deploy

Deployed on [Vercel](https://vercel.com).
